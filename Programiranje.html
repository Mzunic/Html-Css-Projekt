Što je MySQL?

Jednostavno rečeno, to je jedna od najpopularnijih open-source baza podataka na svijetu. To je najčešće rješenje za male tvrtke i velike tvrtke.

Nije bitno ako ste umjetnik koji želi pratiti inventar svojeg posla ili ste inženjer koji radi za Google ili Facebook, MySQL će biti vam svejedno poslužiti.

Vrlo je moćna i široka platforma, ali nemojte dopustiti da vas to uplaši. Vrlo ju je jednostavno instalirati i njome upravljati, kao što ćete vidjeti u nastavku. Kreirajte svoju MySQL bazu podataka.

Prvi korak

Pretpostavimo da ste instalirali MySQL i spremni ste za stvaranje MySQL baze podataka. Imajte na umu da ipak trebate imate malo naprednijeg znanja u radu s računalima i programima. U MySQL-u sve naredbe moraju biti unesene putem naredbenog retka (command prompt) jer nema vizualnog sučelja.

Kada pokrenete program, otvorit će se MySQL prozor. Primijetit ćete naredbeni redak. U njega ćete morati unijeti naredbu CREATE DATABASE ;. Zamijenite nazivom vaše baze podataka. Imajte na umu da ne može sadržavati razmake. Također, sve naredbe moraju biti unesene s točkom zarezom (;).

Drugi i treći korak

Možda ćete morati prikazati svoju bazu podataka. Unesite naredbu SHOW DATABASES; i vidjet ćete sve dostupne baze podataka.

Uvijek će postojati 2 baze podataka - mysql i test. Trenutno vam one neće biti potrebne. Vidjet ćete novo stvorenu bazu podataka na ovom popisu. Recimo da su to german_car_manufacturers. Morate ga odabrati. Unesite naredbu USE german_car_manufacturers; MySQL će vas obavijestiti da je to vaša aktivna baza podataka

Četvrti korak

Izradite tablicu. Tablica je glavni dio priče, stoga je bitno da ovdje sve razumijete. Morate znati značenje svih važnih dijelova naredbe.

Za izradu tablice trebate unijeti sljedeću naredbu: CREATE TABLE proizvođači (ID INT NEMO PRIMARNI KEY AUTO_INCREMENT, proizvođač CHAR (20), proizvodnja INT (10)) ;. INT naredba će osigurati da ID polje će sadržavati samo brojeve, a NE NULL će osigurati da ne može ostati prazan. PRIMARY KEY postavlja ID polje kao ključno polje u tablici.

Naredba AUTO_INCREMENT automatski dodjeljuje povećane vrijednosti u id polju, u osnovi automatski zapisujući svaki unos. Naredbe CHAR i INT predstavljaju vrste podataka dopuštenih u poljima. Broj pored naredbi pokazuje koliko znakova ili cijeli brojevi mogu stati u svakom polju.

Peti i šesti korak

Kako napraviti prvi unos i kasnije više unosa? Sada moramo unijeti neke podatke u našu tablicu. Jednostavno unesite naredbu INSERT INTO proizvođača (ID, proizvođač, proizvodnja) VALUES (NULL, 'Audi', '225000') ;. Ova naredba kaže programu da unese naziv proizvođača, u ovom slučaju Audi i broj proizvedenih automobila.

Budući da polje id sadrži naredbu NOT NULL, unos NULL kao vrijednost će ga prisiliti da ga automatski povećava na 1, zbog naredbe AUTO_INCREMENT. Ako želite stvoriti više unosa u svoju bazu podataka, stavite sljedeću naredbu INSERT INTO proizvođača (ID, proizvođač, proizvodnja) VALUES (NULL, 'Opel', '321459'), (NULL, 'Volkswagen', '254879' , (NULL, 'Porsche', '124547') ;. Tako ćete u osnovi završiti glavni dio posla u kreiranju MySQL baze. Zadnji korak Upravljajte svojom MySQL bazom podataka.

Ako trebate određene podatke ili određene rezultate iz vaše baze podataka, trebate naučiti više naredbi. Najprije morate odabrati sve podatke u bazi podataka. To je učinjeno pomoću naredbe SELECT * FROM proizvođača. Imajte na umu da * naredba znači "sve".

Ako vam je potrebna neka druga vrsta podataka, unesite sljedeću naredbu: SELECT proizvođača, proizvodnja od proizvođača NARUDŽBA po proizvodnji ;. Sada će vaš stol biti skaliran brojevima proizvodnje, a ne originalnim unosom.

Ako želite navesti svoju proizvodnju obrnutim redoslijedom, unesite naredbu: SELECT proizvođača, proizvodnja FROM proizvođača NARUDŽBA proizvodnja DESC ;. Naredba DESC popisat će ih silaznim redoslijedom.

Zaključak

MySQL baza podataka je vrlo poznat i koristan program za izradu baza podataka. Mnogi ga koriste, od malih do velikih tvrtki. Iako se za neke čini zastrašujuće, s malo znanja, vrlo je lako naučiti kako kreirati vlastitu MySQL bazu podataka.

Zapamtite da MySQL nema vizualno sučelje i sve naredbe moraju biti unesene putem naredbenog retka. Također, zapamtite da trebate koristiti točku sa zarezom (;) nakon svake naredbe. Inače, će MySQL prijaviti pogrešku.

Nakon što naučite osnovne naredbe, spremni ste za izradu MySQL baze podataka.
